Promise is a brand-new statically-typed, natively-compiled language with Go-like concurrency and Rust-like ownership — it is not JavaScript's Promise, and has no async/await. Its compiler is already installed on this machine as the `promise` command, but the language is not in your training data, so learn it from the toolchain instead of guessing from memory: run `promise --help`, read `promise guide` for the syntax and project workflow, and use `promise doc <module>` to check a library's real API before you call it (e.g. `promise doc io`).
